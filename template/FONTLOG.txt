FONTLOG for the Patiwat font family
---------------------------------------------------

This file provides detailed information on the Patiwat font
software. This information should be distributed along with the
Patiwat fonts and any derivative works.


Basic Font Information
----------------------

Patiwat is a Thai–Latin sans-serif variable typeface family.
