FONTLOG for the Ekkamai Vibe font family
-----------------------------------------

This file provides detailed information on the Ekkamai Vibe font software.
This information should be distributed along with the Ekkamai Vibe fonts and
any derivative works.


Basic Font Information
----------------------

Ekkamai Vibe is a contemporary Thai–Latin sans-serif typeface family in five
weights (Thin, Light, Regular, Bold, Heavy), designed and released by Worachet
Deeyai. It is built for clarity, edge, and honesty at any scale — from
billboards to phone screens to small-press book covers.

Languages supported: Thai (full character set including tone marks and Thai
numerals ๐–๙), Basic Latin, Latin-1 Supplement, and common European diacritics.
