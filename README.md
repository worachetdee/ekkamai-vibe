# Ekkamai Vibe

A contemporary Thai–Latin sans-serif typeface family in five weights, designed and released by **Ekkamai Foundry**.

Ekkamai Vibe is built for a generation that wants to speak with clarity, edge, and honesty — whether on a billboard above the expressway, a phone screen at midnight, or the cover of a small-press book down a Bangkok side street.

---

## Family

The Ekkamai Vibe family includes 5 weights:

| Weight | File |
|--------|------|
| Thin    | `EkkamaiVibe-Thin.ttf` |
| Light   | `EkkamaiVibe-Light.ttf` |
| Regular | `EkkamaiVibe-Regular.ttf` |
| Bold    | `EkkamaiVibe-Bold.ttf` |
| Heavy   | `EkkamaiVibe-Heavy.ttf` |

---

## Language Support

- Thai (full character set, including tone marks and Thai numerals ๐–๙)
- Basic Latin (A–Z, a–z, 0–9)
- Latin-1 Supplement and common European diacritics
- Standard punctuation and currency symbols

---

## Installation

**macOS** — Double-click the font file and click *Install Font*, or drop into `~/Library/Fonts`.
**Windows** — Right-click the font file and select *Install*, or drop into `C:\Windows\Fonts`.
**Linux** — Copy to `~/.local/share/fonts/` and run `fc-cache -fv`.
**Web** — Use the included [`fonts.css`](fonts.css) with `@font-face`, or convert to WOFF2 for smaller file sizes.
